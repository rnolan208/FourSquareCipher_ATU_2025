Four Square Cipher
📌 Overview

Four Square Cipher is a Java-based encryption tool developed as part of a Software Development course at Atlantic Technological University (ATU).

The project implements the classical Four-Square Cipher to encrypt and decrypt text from files and URLs. It provides a terminal-based interface that allows users to interact with various cryptographic features in a structured and user-friendly way.

🚀 Features

🔐 Encrypt & Decrypt Files

Upload text files for encryption or decryption using the Four-Square Cipher.

🔑 Key Generation

Generate cipher keys required for the Four-Square algorithm.

🔒 Key Encryption

Secure generated keys using a Vigenère Cipher.

📂 File Export

Download:

Encrypted/decrypted text files

Cipher keys (encrypted or plain)

📥 Key Import

Upload previously generated keys for reuse in encryption/decryption.

🎨 Visual Cipher Output

Display the Four-Square Cipher grid in a coloured terminal format using loops.

🧾 Cipher Layout Display

Print the structure/layout of the Four-Square Cipher.

ℹ️ Information Menu

Built-in help section explaining all program features and functionality.

🛠️ Technologies Used

Java

Terminal/Command Line Interface (CLI)

📖 How It Works

The Four-Square Cipher is a classical encryption technique that uses four 5x5 matrices to transform plaintext into ciphertext. This implementation enhances the traditional approach by:

Allowing dynamic key generation

Supporting file-based input/output

Adding an extra layer of security via Vigenère key encryption

Providing visual and interactive terminal outputs
